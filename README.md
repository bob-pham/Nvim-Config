# My Neovim Config

- Inspired by Nvchad, lightly inspired by kickstart.nvim

# Install
```shell
git clone https://github.com/bob-pham/NeovimConfig "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```

# Uninstall
```shell
# Linux / Macos (unix)
rm -rf ~/.config/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.local/share/nvim
```
